ASRock Z490M-ITX/ac
===================

What Works:
- Metal
- Apple Store
- Ethernet (intel 1219v)
- Sleep/Wake
- Sound - Headphones (front), Internal Speaker (rear), Microphone (rear)  (via FakePCIID patch kext).
- CPU/GPU identified correctly in System Properties
- WiFi and Bluetooth (BCM94352Z) 

What doesn't yet
- Ethernet (2.5Gb)
- DRM
- Handoff/Continuity
